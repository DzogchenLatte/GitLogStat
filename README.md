# GitLogStat
bashrc entry to summarize git stat across multiple commits

#Usage instructions
Paste .txt file in your ~/.bashrc file, replace "Andrew Decker" with your name, and run
```
source ~/.bashrc
```
Now you can run:
```
mylogstat commit1 commit2
```
or
```
logstat "Other Author" commit1 commit2
```
Cheers :)
